## Descrição do projeto

Projeto onde o usuario diz qual o nome e o valor da ação( dispesa ou receita) e o projeto faz o balanço descrevendo em uma tabela quais são essas ações e o valor de cada uma.

## Tecnologia Usadas:

-`JavaScript`
-`CSS`
-`React`
